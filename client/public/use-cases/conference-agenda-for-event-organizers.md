# QR Conference Agenda & Session Flows

**Audience:** Event Organizers & Conferences  
**Template:** https://givelive.app/blog/conference-agenda  
**Website:** https://givelive.app/use-cases/conference-agenda-for-event-organizers

> The conference program that updates itself

Give attendees a live mobile agenda via QR code — session selection, room changes, and speaker info without printing a single program.

## How to launch

1. Open https://givelive.app/use-cases/conference-agenda-for-event-organizers
2. Start from the free template at https://givelive.app/blog/conference-agenda
3. Customize copy, branding, and journey steps in the visual builder
4. Publish and place your QR code where your audience already is
